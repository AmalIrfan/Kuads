- [BUILD](./BUILD.md)

To see how to build the project, see [BUILD](./BUILD.md).

## Controls

- <kbd>WASD</kbd>: Move cursor
- <kbd>QWEZ</kbd>: Move cursor diagonally
- <kbd>Shift</kbd>: Set pixel
- <kbd>Ctrl</kbd>: Clear pixel
